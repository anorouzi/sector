Sector Tools

This section describes in detail the Sector tools. These tools are located in ./codeblue2/client/tools.

Method Fuctionality
download accept a connection.
ls assign a local name to an unnamed udt socket.
mkdir release the complete UDT library.
rm close the opened UDT entity and shutdown the connection.
stat connect to the server or the peer side.
sysinfo retrieve last UDT error in the current thread.
upload read the address of the peer side of the connection
