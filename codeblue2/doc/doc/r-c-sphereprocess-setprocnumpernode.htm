setProcNumPerNode

The setProcNumPerNode function specifies the number of SPEs per node.

int setProcNumPerNode
(
   int num
);
Parameters
num
[in] number of SPEs per node.
Return Value

On success, the function returns 1. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

By default, each Slave node only starts one SPE to process data. This function can request multiple SPEs to be started. Using this can speed up Sphere process without inter-node data movement.
