Reference: Tools

download

The download tool downloads one or more files from the Sector system.

download IP PORT FILELIST LOCALPATH
Parameters
IP
The IP address or the domain name of the Sector master server .
PORT
The port number of the Sector master server.
FILELIST
A text file that contains the list of files to be downloaded, one per line.
LOCALPATH
The local directory to store the downloaded files.
Description

The command uses account "test".
