Reference: Tools

rm

The rm tool deletes a file or a directory.

download IP PORT PATH
Parameters
IP
The IP address or the domain name of the Sector master server .
PORT
The port number of the Sector master server.
PATH
The path to the file or the directory to be deleted.
Description

The command uses account "test".

All contents in the directory, including sub-directories, will be deleted. There is no way to restore the deleted files.
